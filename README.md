# # "Сервис TAXI".
Роли:
- Водитель;
- Пассажир;
- Внутренний специалист сервиса.
# Возможности участников в приложении:
- Водитель: арендует машину, снимает с брони, осуществляет заправку автомобиля.
- Пассажир: совершает заказ.
- внутренний специалист занимается выгрузкой и аналитическими подсчетами.
# Возможности системы:
- назначать водителя;
- менять статус заказа;
- вести подсчет оплаты с коэффициентом на конкретное время.
